# Reset Session

A brief reset between tasks or after cognitive overload.

## Quick Bell Reset

```
/bell
```

One strike. Natural decay. Done.

## Overloaded Reset

```
/ear +mood:overloaded
```

Sparse bell reset with low stable drone. 5 minutes default.

## Tired Wind-Down

```
/ear +mood:tired
```

Minimal low room tone with sparse granular air.

## Short Silence

```
/still 1
```

One minute. Timer only. No audio. Return to work.
