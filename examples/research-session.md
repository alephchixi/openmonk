# Research Session

Extended session for reading, research, or analysis.

## Deep Research

```
/zen 60
```

One hour of structured silence. Bell at start and end.

## Background Drone

```
/om 60 --low --sparse --far --breathy
```

Low ambient drone for extended reading sessions.

## Foggy Start

```
/ear +mood:foggy
```

Thin distant drone to ease into a session when the mind is unfocused.
