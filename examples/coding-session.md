# Coding Session

A focused coding session using OpenMonk for structural silence.

## Setup

```
/zen 30
```

## Flow

1. Begin session → bell rings
2. Silent timer starts: 30 minutes
3. Work. No audio. No interruption.
4. Timer completes → end bell rings
5. Session complete.

## Variant: With Drone

```
/om 30 --low --sparse --far
```

Sustained low drone as ambient background for deep work.

## Variant: Quick Reset

```
/bell
```

Single bell to mark a transition between tasks.
