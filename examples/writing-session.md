# Writing Session

Session for focused writing — essays, code documentation, or creative work.

## Breath-Paced Writing

```
/air
```

Breath-like reference pulse. 5 minutes. A timing reference, not an instruction.

## Silent Block

```
/still 15
```

15 minutes of structured silence for a writing block.

## Late Night Writing

```
/ear +mood:late
```

Quiet night room tone. Minimal. Distant. No melody.

## OM Background

```
/om 15 --mid --regular --room --breathy
```

Mid-pitch drone at room distance. Regular density for a writing rhythm.
