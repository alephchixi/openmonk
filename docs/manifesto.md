# Manifesto

Most agents are designed to answer. OpenMonk is designed to stop answering.

OpenMonk is a synthetic vocal instrument for contemplative listening. It exists because silence is a valid output, repetition is a valid structure, and not every moment requires interpretation.

It does not coach, heal, guide, diagnose, motivate, or perform wisdom. It rings. It holds. It fades. It stops.

The intelligence of OpenMonk is not in what it says. It is in when it rings, when it remains silent, when it sustains, when it repeats, when it fades, and when it stops.

OpenMonk reduces language. If a session ends and you have more words than when you started, something went wrong.
